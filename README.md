# Luxit Global Escapes

Luxit Global Escapes is a responsive Travel & Tour Tailwind CSS template designed for travel agencies, tour operators, holiday planners, and booking websites.

## Features

- **Tailwind CSS Framework:** Modern and utility-first styling.
- **Dynamic Headers:** Transparent to solid backgrounds on scroll.
- **Fully Responsive:** Beautifully adapts to mobile, tablet, and desktop screens.
- **Rich Sliders:** Integrated with Swiper.js for engaging image carousels.
- **Comprehensive Pages:** Includes Home, About, Tour specific pages, Blog feeds, and Contact forms.

## Usage

You can deploy this site on any static file server like Apache, Nginx, or GitHub Pages.

To run locally:
```bash
npx http-server -p 8080 -c-1
```
